# Flashing HTML HUD Fix for CS2

A CounterStrikeSharp plugin that fixes the annoying flashing/flickering HTML HUD issue caused by the Armory/AG2 updates in Counter-Strike 2

## Features
* Prevents HTML HUD elements from flashing during round restarts
* Safely handles map changes (e.g., automated transitions from Dust2 to main maps) without breaking the HUD state

## Installation
1. Ensure you have [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp) installed on your server
2. Download the latest `FlashingHtmlHudFix` files
3. Place the plugin folder into `game/csgo/addons/counterstrikesharp/plugins/`
4. Restart your server or change the map
