FlashingHTMLHUDFix for CS2

A CounterStrikeSharp plugin that fixes the annoying flashing/flickering HTML HUD issue caused by the Armory/AG2 updates in Counter-Strike 2.

Features
Prevents HTML HUD elements from flashing during round restarts.

Installation
Ensure you have CounterStrikeSharp installed on your server.
Download the latest FlashingHtmlHudFix files.
Place the plugin folder into game/csgo/addons/counterstrikesharp/plugins/.
Restart your server or change the map.
